# 🦅 Guía de Captura Manual: El Oráculo de Nueva Eternia

Esta guía explica cómo importar ofertas de Wallapop al Oráculo de Nueva Eternia usando el sistema de importación manual asistida.

---

## 🛠️ ¿Cómo funciona?

Debido a los bloqueos de seguridad de Wallapop que impiden el scraping automático continuo, hemos implementado el **Wallapop Manual Importer**.

1. **Exploración**: Buscas MOTU Origins en Wallapop desde tu navegador habitual.
2. **Captura**: Copias el texto del anuncio o la URL directa.
3. **Infiltración**: Pegas los datos en el apartado **Wallapop** del panel de **Configuración**.
4. **Procesamiento**: El sistema extrae el precio, título e imagen y lo deposita en el Purgatorio para su validación definitiva, marcándolo automáticamente como tipo `Peer-to-Peer`.
5. **Refugio de Valor (Teoría de Cuarentena)**: Las ofertas capturadas en este mercado secundario se miden contra el **Percentil 25** (Market Floor). Su precio queda aislado en el circuito P2P, lo que asegura que estos remates u ofertas infravaloradas no causen deflación en el valor real de tasación de las piezas en "Mi Legado".

## 🚀 Pasos para la Incursión

1. Abre Wallapop y busca la figura que deseas.
2. Copia la URL del anuncio.
3. Ve al panel de **Configuración** en el Oráculo.
4. Selecciona la pestaña **Wallapop**.
5. Pega el enlace y pulsa **"Procesar Hallazgo"**.

---

> [!TIP]
> Si el sistema no detecta automáticamente la imagen, puedes subirla manualmente desde el mismo panel de importación.
