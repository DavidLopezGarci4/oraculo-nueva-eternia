# Skill de Auditoría y Roadmap (3OX)

## Auditoría Inicial
- Al iniciar o retomar el proyecto, realiza un escaneo obligatorio (`ls -R`) para mapear archivos.
- Actualiza `architecture_map.md` con las dependencias detectadas.

## Roadmap Dinámico (SMART)
- Cada iteración debe proponer mejoras en `MASTER_ROADMAP.md`.
- Las tareas deben ser: Específicas, Medibles, Alcanzables, Relevantes y Temporales.

## Trazabilidad de Cambios
- Mantener `LOG_MAESTRO_NUEVA_ETERNIA` (o el log definido) actualizado tras cada ciclo operativo.
